Neste desafio, você terá a oportunidade de criar seu primeiro projeto lógico de 
banco de dados utilizando o MySQL. O objetivo é replicar a modelagem de um banco 
de dados para um cenário de e-commerce. Prepare-se para aplicar seus conhecimentos 
em modelagem de banco de dados e traduzir os requisitos do cenário em uma estrutura 
lógica coerente usando o MySQL.
